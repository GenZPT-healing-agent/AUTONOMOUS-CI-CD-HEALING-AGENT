## 🎯 User Journey

```
1. User visits your site → Lands on modern hero page (/)
2. Clicks "Initialize Agent" → Goes to terminal (/terminal)
3. Fills in repo details → Clicks "Run Agent"
4. Agent executes → Navigates to dashboard (/dashboard)
5. Views results → Can navigate back via nav bar
```
